# AWS Amplify Gen 2: Python-runtime Lambda Function

This project demonstrates how to implement and use Python-runtime Lambda functions in AWS Amplify Gen 2 applications with Next.js.

## Features

-   Next.js application setup
-   AWS Amplify Gen 2 integration
-   Python-runtime Lambda function implementation
-   Frontend-backend connection via GraphQL API
-   Simple user interface with Amplify UI components

## Prerequisites

-   Node.js (v18.x or later)
-   npm (v9.x or later)
-   AWS account
-   AWS Amplify CLI
-   Python 3.12

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/superluminal-jp/amplify-gen2-python-runtime-lambda
cd amplify-gen2-python-runtime-lambda
```

2. Install dependencies:

```bash
npm install
```

3. Deploy the backend:

```bash
npx ampx sandbox
```

4. Start the development server:

```bash
npm run dev
```

5. Visit http://localhost:3000 to view the application
